# Othello Game

## Description

This project is a Python implementation of the board game **Othello** that runs in the terminal. It allows two players to take turns placing pieces on the board, validates legal moves, flips captured pieces, tracks the board state, and determines the winner when the game ends.

## Tools Used

* **Python**
* **VS Code**
* **Terminal / Command Line**
* **GitHub**

## Skills Demonstrated

* Object-oriented programming with classes and methods
* Game logic implementation
* 2D list manipulation for board management
* Input validation
* Conditional logic and looping
* Problem solving and debugging
* Command-line program development

## How to Play

1. Open the project folder in **VS Code**.
2. Open the terminal.
3. Run the program with:

```bash
python othello.py
```

If needed on Windows, use:

```bash
py othello.py
```

4. Enter the name for the black player.
5. Enter the name for the white player.
6. Follow the prompts to enter the **row** and **column** for each move.
7. The game ends when neither player has a valid move left.
8. The program prints the final board and announces the winner.

## Board Symbols

* `*` = board boundary
* `.` = empty space
* `X` = black piece
* `O` = white piece
